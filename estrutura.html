Projeto Prático

Durante o curso, você desenvolverá um projeto prático completo: um sistema de gerenciamento para estúdio de pilates.

Visão Geral do Projeto

O projeto consiste em um sistema de gerenciamento para estúdio de pilates, com funcionalidades de cadastro de alunos, agendamento de aulas, controle de pagamentos e relatórios.

O foco principal será a implementação de um CRUD básico para o cadastro de alunos, utilizando Node.js e Express, com integração a banco de dados MySQL.

Ver detalhes do projeto

Etapas de Desenvolvimento

  • Configuração do ambiente de desenvolvimento
  • Estruturação do projeto
  • Modelagem do banco de dados
  • Implementação do CRUD de alunos
  • Validação e tratamento de erros
  • Testes e documentação
