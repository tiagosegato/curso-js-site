Módulo 1: Lógica de Programação com JavaScript

Fundamentos essenciais para iniciar no desenvolvimento com JavaScript

Visão Geral do Módulo

Neste módulo, você aprenderá os fundamentos da lógica de programação utilizando JavaScript como linguagem base. Serão abordados conceitos essenciais para o desenvolvimento de software, desde variáveis e tipos de dados até estruturas de controle e funções.

Exemplos Práticos

Calculadora Simples

// Função para realizar operações matemáticas básicas
function calculadora(num1, num2, operacao) {
  // Validação de entrada
  if (typeof num1 !== 'number' || typeof num2 !== 'number') {
    throw new Error('Os valores informados devem ser números');
  }
  
  // Execução da operação solicitada
  switch (operacao) {
    case 'soma':
      return num1 + num2;
    case 'subtracao':
      return num1 - num2;
    case 'multiplicacao':
      return num1 * num2;
    case 'divisao':
      if (num2 === 0) {
        throw new Error('Não é possível dividir por zero');
      }
      return num1 / num2;
    default:
      throw new Error('Operação inválida. Use: soma, subtracao, multiplicacao ou divisao');
  }
}

// Exemplos de uso
try {
  console.log('Soma: ' + calculadora(10, 5, 'soma')); // 15
  console.log('Subtração: ' + calculadora(10, 5, 'subtracao')); // 5
  console.log('Multiplicação: ' + calculadora(10, 5, 'multiplicacao')); // 50
  console.log('Divisão: ' + calculadora(10, 5, 'divisao')); // 2
  
  // Teste com erro
  console.log(calculadora(10, 0, 'divisao')); // Erro: Não é possível dividir por zero
} catch (erro) {
  console.error('Erro: ' + erro.message);
}

Manipulação de Arrays

// Array de alunos
const alunos = [
  { id: 1, nome: 'João', nota: 8.5 },
  { id: 2, nome: 'Maria', nota: 9.0 },
  { id: 3, nome: 'Pedro', nota: 7.5 },
  { id: 4, nome: 'Ana', nota: 6.8 },
  { id: 5, nome: 'Carlos', nota: 8.0 }
];

// Filtrando alunos aprovados (nota >= 7)
const alunosAprovados = alunos.filter(aluno => aluno.nota >= 7);
console.log('Alunos aprovados:');
console.table(alunosAprovados);

// Calculando a média das notas
const somaNotas = alunos.reduce((acumulador, aluno) => acumulador + aluno.nota, 0);
const mediaNotas = somaNotas / alunos.length;
console.log(`Média das notas: ${mediaNotas.toFixed(2)}`);

// Encontrando o aluno com a maior nota
const alunoMaiorNota = alunos.reduce((maior, atual) => 
  atual.nota > maior.nota ? atual : maior
);
console.log(`Aluno com maior nota: ${alunoMaiorNota.nome} (${alunoMaiorNota.nota})`);

// Adicionando um novo aluno
alunos.push({ id: 6, nome: 'Juliana', nota: 9.5 });

// Ordenando alunos por nota (decrescente)
const alunosOrdenados = [...alunos].sort((a, b) => b.nota - a.nota);
console.log('Alunos ordenados por nota:');
console.table(alunosOrdenados);

Exercícios Práticos

Para fixar os conceitos aprendidos neste módulo, recomendamos a realização dos seguintes exercícios:

  1. Crie uma função que verifica se um número é par ou ímpar.
  2. Desenvolva um programa que calcule a média de um array de números.
  3. Implemente uma função que inverta uma string.
  4. Crie um programa que conte a ocorrência de cada palavra em uma frase.
  5. Desenvolva uma função que verifique se uma palavra é um palíndromo.
  6. Implemente um conversor de temperatura (Celsius para Fahrenheit e vice-versa).
  7. Crie uma função que encontre o maior e o menor valor em um array de números.
  8. Desenvolva um programa que simule um jogo de adivinhação de números.

Você encontrará mais exercícios e desafios na seção de Exercícios.
