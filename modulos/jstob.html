Módulo 3: JavaScript e Orientação a Objetos

Aplicando conceitos de Programação Orientada a Objetos com JavaScript

Visão Geral do Módulo

Neste módulo, você aprenderá a aplicar os conceitos de Programação Orientada a Objetos (POO) utilizando JavaScript. A POO é um paradigma de programação que organiza o código em torno de objetos, permitindo criar aplicações mais modulares, reutilizáveis e fáceis de manter.

Exemplos Práticos

Classes e Herança em JavaScript

// Classe base Pessoa
class Pessoa {
  constructor(nome, idade) {
    this.nome = nome;
    this.idade = idade;
  }
  
  apresentar() {
    return `Olá, meu nome é ${this.nome} e tenho ${this.idade} anos.`;
  }
  
  // Método estático
  static criarPessoa(nome, idade) {
    return new Pessoa(nome, idade);
  }
}

// Classe Aluno que herda de Pessoa
class Aluno extends Pessoa {
  constructor(nome, idade, matricula, curso) {
    // Chama o construtor da classe pai
    super(nome, idade);
    
    this.matricula = matricula;
    this.curso = curso;
  }
  
  // Sobrescreve o método apresentar
  apresentar() {
    return `${super.apresentar()} Sou aluno(a) do curso de ${this.curso} com matrícula ${this.matricula}.`;
  }
  
  estudar() {
    return `${this.nome} está estudando ${this.curso}.`;
  }
}

// Classe Professor que herda de Pessoa
class Professor extends Pessoa {
  constructor(nome, idade, especialidade, departamento) {
    super(nome, idade);
    
    this.especialidade = especialidade;
    this.departamento = departamento;
  }
  
  apresentar() {
    return `${super.apresentar()} Sou professor(a) de ${this.especialidade} no departamento de ${this.departamento}.`;
  }
  
  ensinar() {
    return `${this.nome} está ensinando ${this.especialidade}.`;
  }
}

// Criando instâncias
const pessoa = new Pessoa('Carlos', 30);
const aluno = new Aluno('Maria', 22, '2023001', 'Engenharia de Software');
const professor = new Professor('João', 45, 'JavaScript', 'Tecnologia da Informação');

console.log(pessoa.apresentar());
// Saída: Olá, meu nome é Carlos e tenho 30 anos.

console.log(aluno.apresentar());
// Saída: Olá, meu nome é Maria e tenho 22 anos. Sou aluno(a) do curso de Engenharia de Software com matrícula 2023001.

console.log(professor.apresentar());
// Saída: Olá, meu nome é João e tenho 45 anos. Sou professor(a) de JavaScript no departamento de Tecnologia da Informação.

console.log(aluno.estudar());
// Saída: Maria está estudando Engenharia de Software.

console.log(professor.ensinar());
// Saída: João está ensinando JavaScript.

// Usando método estático
const novaPessoa = Pessoa.criarPessoa('Ana', 28);
console.log(novaPessoa.apresentar());
// Saída: Olá, meu nome é Ana e tenho 28 anos.

Encapsulamento com Propriedades Privadas

// Usando propriedades privadas (ES2020+)
class ContaBancaria {
  // Propriedades privadas com #
  #saldo;
  #senha;
  #extrato;
  
  constructor(titular, saldoInicial, senha) {
    this.titular = titular;
    this.#saldo = saldoInicial;
    this.#senha = senha;
    this.#extrato = [];
    
    // Registra a criação da conta
    this.#registrarTransacao('Abertura de conta', saldoInicial);
  }
  
  // Método privado
  #registrarTransacao(tipo, valor) {
    const data = new Date();
    this.#extrato.push({
      data: data.toISOString(),
      tipo,
      valor
    });
  }
  
  // Método privado para validar senha
  #validarSenha(senha) {
    return this.#senha === senha;
  }
  
  // Métodos públicos
  depositar(valor) {
    if (valor <= 0) {
      throw new Error('O valor do depósito deve ser positivo');
    }
    
    this.#saldo += valor;
    this.#registrarTransacao('Depósito', valor);
    return `Depósito de R$ ${valor.toFixed(2)} realizado com sucesso`;
  }
  
  sacar(valor, senha) {
    if (!this.#validarSenha(senha)) {
      throw new Error('Senha incorreta');
    }
    
    if (valor <= 0) {
      throw new Error('O valor do saque deve ser positivo');
    }
    
    if (valor > this.#saldo) {
      throw new Error('Saldo insuficiente');
    }
    
    this.#saldo -= valor;
    this.#registrarTransacao('Saque', -valor);
    return `Saque de R$ ${valor.toFixed(2)} realizado com sucesso`;
  }
  
  consultarSaldo(senha) {
    if (!this.#validarSenha(senha)) {
      throw new Error('Senha incorreta');
    }
    
    return `Saldo atual: R$ ${this.#saldo.toFixed(2)}`;
  }
  
  consultarExtrato(senha) {
    if (!this.#validarSenha(senha)) {
      throw new Error('Senha incorreta');
    }
    
    return this.#extrato;
  }
  
  // Getters e Setters
  get saldo() {
    return 'Acesso negado. Use o método consultarSaldo(senha)';
  }
  
  set saldo(valor) {
    throw new Error('Não é possível alterar o saldo diretamente');
  }
}

// Uso da classe
try {
  const conta = new ContaBancaria('João Silva', 1000, '1234');
  
  console.log(conta.depositar(500));
  // Saída: Depósito de R$ 500.00 realizado com sucesso
  
  console.log(conta.consultarSaldo('1234'));
  // Saída: Saldo atual: R$ 1500.00
  
  console.log(conta.sacar(200, '1234'));
  // Saída: Saque de R$ 200.00 realizado com sucesso
  
  console.log(conta.consultarSaldo('1234'));
  // Saída: Saldo atual: R$ 1300.00
  
  // Tentativa de acesso direto às propriedades privadas
  console.log(conta.saldo);
  // Saída: Acesso negado. Use o método consultarSaldo(senha)
  
  // Tentativa de alteração direta do saldo
  // conta.saldo = 5000; // Erro: Não é possível alterar o saldo diretamente
  
  // Tentativa de saque com senha incorreta
  // console.log(conta.sacar(100, '4321')); // Erro: Senha incorreta
  
  // Consulta do extrato
  console.log(conta.consultarExtrato('1234'));
  // Saída: Array com as transações realizadas
  
} catch (erro) {
  console.error(`Erro: ${erro.message}`);
}

Exercícios Práticos

Para fixar os conceitos aprendidos neste módulo, recomendamos a realização dos seguintes exercícios:

  1. Crie uma classe Retangulo com propriedades para largura e altura, e métodos para calcular área e perímetro.
  2. Implemente uma hierarquia de classes para um sistema de e-commerce (Produto, ProdutoFisico, ProdutoDigital).
  3. Desenvolva um sistema de gerenciamento de biblioteca com classes para Livro, Autor, Usuario e Emprestimo.
  4. Crie uma classe ContaBancaria com propriedades privadas e métodos para depósito, saque e consulta de saldo.
  5. Implemente o padrão Observer para criar um sistema de notificações.
  6. Desenvolva um sistema de gerenciamento de tarefas utilizando classes e herança.
  7. Crie uma classe Veículo e subclasses para diferentes tipos de veículos (Carro, Moto, Caminhão).
  8. Implemente o padrão Factory para criar diferentes tipos de documentos (PDF, Word, Excel).

Você encontrará mais exercícios e desafios na seção de Exercícios.
