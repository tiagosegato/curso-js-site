Módulo 2: Git, GitHub e GitLab

Controle de versão e colaboração em projetos de desenvolvimento

Visão Geral do Módulo

Neste módulo, você aprenderá sobre sistemas de controle de versão, com foco no Git e nas plataformas GitHub e GitLab. Essas ferramentas são essenciais para o desenvolvimento colaborativo de software, permitindo o gerenciamento eficiente de código-fonte e facilitando o trabalho em equipe.

Exemplos Práticos

Configuração Inicial do Git

# Configuração do nome de usuário
git config --global user.name "Seu Nome"

# Configuração do email
git config --global user.email "seu.email@exemplo.com"

# Verificação das configurações
git config --list

Fluxo Básico de Trabalho com Git

# Inicialização de um repositório
git init

# Verificação do status
git status

# Adição de arquivos ao staging area
git add arquivo.js
git add .  # Adiciona todos os arquivos modificados

# Criação de um commit
git commit -m "Adiciona funcionalidade X"

# Visualização do histórico
git log --oneline

# Criação de uma branch
git branch feature/nova-funcionalidade

# Mudança para a nova branch
git checkout feature/nova-funcionalidade

# Criação e mudança para uma nova branch (atalho)
git checkout -b feature/outra-funcionalidade

# Mesclagem de branches
git checkout main
git merge feature/nova-funcionalidade

# Envio para repositório remoto
git push origin main

Exercícios Práticos

Para fixar os conceitos aprendidos neste módulo, recomendamos a realização dos seguintes exercícios:

  1. Configure o Git em sua máquina local com seu nome e email.
  2. Crie um repositório local e adicione alguns arquivos de exemplo.
  3. Crie uma conta no GitHub (se ainda não tiver) e crie um repositório remoto.
  4. Conecte seu repositório local ao repositório remoto e envie seus commits.
  5. Crie uma branch para uma nova funcionalidade, faça algumas alterações e mescle com a branch principal.
  6. Simule um conflito de merge e resolva-o.
  7. Crie um fork de um projeto open source e faça um pull request com uma melhoria ou correção.
  8. Configure um projeto com GitHub Pages.

Você encontrará mais exercícios e desafios na seção de Exercícios.
