Projeto Prático

Sistema de Gerenciamento para Estúdio de Pilates

Visão Geral do Projeto

O projeto prático do curso consiste no desenvolvimento de um sistema de gerenciamento para um estúdio de pilates, com foco no cadastro e gerenciamento de alunos. Este projeto foi especialmente projetado para permitir que você aplique os conhecimentos adquiridos ao longo do curso em um contexto real e prático.

Público-alvo

Estúdios de pilates e academias

Tecnologias

Node.js, Express, MySQL, Sequelize

Complexidade

Iniciante a Intermediário

Objetivos do Projeto

O principal objetivo deste projeto é proporcionar uma experiência prática de desenvolvimento backend com JavaScript, aplicando os conceitos aprendidos durante o curso. Ao concluir o projeto, você será capaz de:

  • Estruturar um projeto Node.js seguindo boas práticas
  • Implementar uma API RESTful com Express
  • Criar e manipular um banco de dados relacional
  • Utilizar um ORM (Sequelize) para interagir com o banco de dados
  • Implementar operações CRUD (Create, Read, Update, Delete)
  • Aplicar conceitos de programação orientada a objetos
  • Utilizar padrões de desenvolvimento de software
  • Implementar validações e tratamento de erros

Funcionalidades Principais

O sistema de gerenciamento para estúdio de pilates incluirá as seguintes funcionalidades principais:

Cadastro de Alunos

Permite cadastrar novos alunos no sistema, com informações como nome, email, telefone e data de nascimento.

Listagem de Alunos

Exibe a lista de todos os alunos cadastrados no sistema, com opções de filtragem e ordenação.

Busca de Alunos

Permite buscar alunos específicos por ID, nome ou email, exibindo suas informações detalhadas.

Atualização de Alunos

Permite atualizar as informações de um aluno existente, como telefone, email ou status de ativo/inativo.

Exclusão de Alunos

Permite excluir um aluno do sistema, com opção de exclusão lógica (inativação) ou física.

Nota Importante

O foco inicial do projeto é implementar um CRUD básico para o cadastro de alunos, tornando-o acessível para alunos iniciantes. A modelagem completa do banco de dados inclui outras entidades (como instrutores, modalidades, aulas e pagamentos) que podem ser implementadas em versões futuras do sistema, à medida que você avança no curso e adquire mais conhecimentos.

Estrutura do Projeto

O projeto será desenvolvido seguindo uma estrutura organizada e modular, baseada no padrão MVC (Model-View-Controller):

pilates-studio-api/
├── node_modules/
├── src/
│   ├── config/
│   │   ├── database.js       # Configuração de conexão com o banco de dados
│   │   └── server.js         # Configuração do servidor Express
│   ├── controllers/
│   │   └── alunoController.js # Controlador para operações de alunos
│   ├── models/
│   │   └── Aluno.js          # Modelo de dados para alunos
│   ├── routes/
│   │   ├── index.js          # Rota principal
│   │   └── alunoRoutes.js    # Rotas para operações de alunos
│   ├── middlewares/
│   │   ├── errorHandler.js   # Middleware para tratamento de erros
│   │   └── validator.js      # Middleware para validação de dados
│   ├── utils/
│   │   └── responseFormatter.js # Utilitário para formatação de respostas
│   └── app.js                # Ponto de entrada da aplicação
├── .env                      # Variáveis de ambiente
├── .gitignore                # Arquivos a serem ignorados pelo Git
├── package.json              # Dependências e scripts
└── README.md                 # Documentação do projeto

Tecnologias Utilizadas

O projeto será desenvolvido utilizando as seguintes tecnologias:

Node.js

Ambiente de execução JavaScript server-side

Express

Framework web para Node.js

MySQL

Sistema de gerenciamento de banco de dados relacional

Sequelize

ORM (Object-Relational Mapping) para Node.js
