Exercícios Práticos

Fixe seu conhecimento com exercícios práticos para cada módulo do curso

Exercícios por Módulo

Pratique os conceitos aprendidos em cada módulo com exercícios específicos.

Módulo 1: Lógica de Programação com JavaScript

Exercício 1: Calculadora Simples

Desenvolva uma calculadora simples que realize as quatro operações básicas (adição, subtração, multiplicação e divisão).

Requisitos:

  • Criar funções para cada operação
  • Implementar validação de entrada
  • Tratar divisão por zero
  • Exibir o resultado no console

Exercício 2: Verificador de Palíndromos

Crie uma função que verifique se uma palavra ou frase é um palíndromo (lê-se igual de trás para frente, desconsiderando espaços e pontuação).

Exemplos de palíndromos:

  • "ana"
  • "radar"
  • "socorram-me subi no onibus em marrocos"

Exercício 3: Conversor de Temperatura

Desenvolva um programa que converta temperaturas entre Celsius, Fahrenheit e Kelvin.

Fórmulas de conversão:

  • Celsius para Fahrenheit: (C × 9/5) + 32
  • Celsius para Kelvin: C + 273.15
  • Fahrenheit para Celsius: (F - 32) × 5/9
  • Fahrenheit para Kelvin: (F - 32) × 5/9 + 273.15
  • Kelvin para Celsius: K - 273.15
  • Kelvin para Fahrenheit: (K - 273.15) × 9/5 + 32
Módulo 2: Git, GitHub e GitLab

Exercício 1: Configuração do Git

Configure o Git em sua máquina local e crie um repositório para os exercícios do curso.

Passos:

  1. Instalar o Git (se ainda não estiver instalado)
  2. Configurar nome de usuário e email
  3. Criar um novo repositório local
  4. Adicionar um arquivo README.md
  5. Fazer o primeiro commit

Exercício 2: Trabalhando com Branches

Pratique o trabalho com branches criando uma feature simples e mesclando-a ao branch principal.

Passos:

  1. Criar um branch chamado "feature/calculadora"
  2. Adicionar o código da calculadora do Módulo 1
  3. Fazer commit das alterações
  4. Voltar ao branch principal
  5. Mesclar o branch "feature/calculadora"
  6. Resolver conflitos (se houver)

Exercício 3: GitHub e Colaboração

Crie um repositório no GitHub e pratique o fluxo de trabalho colaborativo.

Passos:

  1. Criar um repositório no GitHub
  2. Conectar o repositório local ao remoto
  3. Enviar os branches locais para o remoto
  4. Simular uma colaboração (criar um branch, fazer alterações, enviar para o remoto)
  5. Criar um pull request
  6. Revisar e mesclar o pull request
Módulo 3: JavaScript e Orientação a Objetos

Exercício 1: Criação de Classes

Crie uma classe Pessoa com atributos e métodos, e depois crie subclasses para tipos específicos de pessoas.

Requisitos:

  • Classe Pessoa com atributos nome, idade, email
  • Métodos para obter e modificar atributos
  • Subclasse Aluno com atributos adicionais (matrícula, curso)
  • Subclasse Professor com atributos adicionais (disciplina, salário)
  • Implementar polimorfismo em um método comum

Exercício 2: Sistema de Biblioteca

Desenvolva um sistema simples de biblioteca utilizando conceitos de OO.

Classes a serem implementadas:

  • Livro (título, autor, ISBN, disponível)
  • Usuário (nome, ID, livros emprestados)
  • Biblioteca (coleção de livros, métodos para emprestar e devolver)

Funcionalidades:

  • Cadastrar livros e usuários
  • Emprestar livros
  • Devolver livros
  • Listar livros disponíveis
  • Listar livros emprestados por usuário

Exercício 3: Implementação de Padrões de Projeto

Implemente um dos padrões de projeto estudados em um contexto prático.

Opções de padrões:

  • Singleton para um gerenciador de configurações
  • Factory para criação de diferentes tipos de documentos
  • Observer para um sistema de notificações
  • Strategy para diferentes métodos de pagamento
Módulo 6: Node.js

Exercício 1: Servidor HTTP Básico

Crie um servidor HTTP básico utilizando o módulo http do Node.js.

Requisitos:

  • Servidor que responda a requisições GET
  • Rota principal que retorne uma mensagem de boas-vindas
  • Rota /info que retorne informações sobre o servidor
  • Tratamento de rotas não encontradas

Exercício 2: API REST com Express

Desenvolva uma API REST simples para gerenciar uma lista de tarefas.

Funcionalidades:

  • Listar todas as tarefas (GET /tarefas)
  • Obter uma tarefa específica (GET /tarefas/:id)
  • Criar uma nova tarefa (POST /tarefas)
  • Atualizar uma tarefa (PUT /tarefas/:id)
  • Excluir uma tarefa (DELETE /tarefas/:id)

Estrutura da tarefa:

  • id: número único
  • titulo: string
  • descricao: string
  • concluida: boolean
  • dataCriacao: date

Exercício 3: Integração com Banco de Dados

Expanda o exercício anterior integrando-o com um banco de dados MySQL.

Requisitos:

  • Configurar conexão com MySQL
  • Criar modelo de Tarefa com Sequelize
  • Implementar operações CRUD utilizando o modelo
  • Adicionar validação de dados
  • Implementar tratamento de erros
Projeto Prático: Sistema de Gerenciamento para Estúdio de Pilates

Exercício 1: Configuração Inicial do Projeto

Configure o ambiente de desenvolvimento e a estrutura básica do projeto.

Tarefas:

  1. Inicializar o projeto com npm
  2. Instalar dependências necessárias
  3. Configurar estrutura de diretórios
  4. Criar arquivo de configuração do servidor
  5. Implementar rota de teste

Exercício 2: Modelagem do Banco de Dados

Crie o modelo de banco de dados para o sistema de gerenciamento do estúdio de pilates.

Tarefas:

  1. Criar banco de dados MySQL
  2. Definir modelo de Aluno
  3. Configurar conexão com o banco de dados
  4. Sincronizar modelo com o banco de dados

Exercício 3: Implementação do CRUD de Alunos

Implemente as operações CRUD para o gerenciamento de alunos.

Tarefas:

  1. Criar controlador de alunos
  2. Implementar rota para listar alunos
  3. Implementar rota para buscar aluno por ID
  4. Implementar rota para criar aluno
  5. Implementar rota para atualizar aluno
  6. Implementar rota para excluir aluno
  7. Adicionar validação de dados
  8. Implementar tratamento de erros

Exercício 4: Testes e Documentação

Teste a API e crie documentação para as rotas implementadas.

Tarefas:

  1. Testar todas as rotas utilizando Postman ou Insomnia
  2. Documentar as rotas da API
  3. Criar README com instruções de instalação e uso
  4. Adicionar comentários explicativos no código
