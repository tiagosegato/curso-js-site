Bibliotecas e Ferramentas

Conheça as principais bibliotecas e ferramentas utilizadas no desenvolvimento backend com JavaScript.

Node.js

Ambiente de execução JavaScript server-side que permite executar código JavaScript fora do navegador.

Site Oficial Documentação

Express.js

Framework web minimalista e flexível para Node.js que fornece um conjunto robusto de recursos para aplicativos web e móveis.

Site Oficial Documentação

Sequelize

ORM (Object-Relational Mapping) baseado em promessas para Node.js, que suporta PostgreSQL, MySQL, SQLite e MSSQL.

Site Oficial Documentação

Jest

Framework de teste JavaScript com foco na simplicidade, que funciona com projetos usando Babel, TypeScript, Node.js, React, Angular e Vue.js.

Site Oficial Documentação

ESLint

Ferramenta de análise de código estático para identificar padrões problemáticos encontrados no código JavaScript.

Site Oficial Documentação

JWT (JSON Web Tokens)

Método compacto e autônomo para transmitir informações com segurança entre as partes como um objeto JSON.

Site Oficial Documentação

Materiais de Estudo

Livros Recomendados

  • Eloquent JavaScript - Marijn Haverbeke

    Uma introdução moderna à programação JavaScript, cobrindo desde os fundamentos até tópicos avançados.

    Disponível online gratuitamente
  • You Don't Know JS - Kyle Simpson

    Série de livros que mergulha profundamente nos mecanismos da linguagem JavaScript.

    Disponível no GitHub
  • Node.js Design Patterns - Mario Casciaro e Luciano Mammino

    Guia prático para implementar padrões de design em aplicações Node.js.

  • Clean Code: A Handbook of Agile Software Craftsmanship - Robert C. Martin

    Princípios e práticas para escrever código limpo e manutenível.
